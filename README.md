# Ogólny opis systemu:
Aplikacja jest zbudowana na zasadzie interfejsu graficznego, na którym Użytkownik może dowolnie rozmieścić Urządzenia, w celu intuicyjnego zarządzania.
Użytkownik jest w stanie uzyskać informacje na temat Urządzeń bezpośrednio z Aplikacji, np. odczytać temperaturę z termometru. Ponad to Użytkownik może zarządzać każdym Urządzeniem z osobna – np. włączyć/wyłączyć żarówkę, zmienić temperaturę ogrzewania, za pomocą Aplikacji.
Dodatkowo możliwe jest też połączenie konkretnych Urządzeń, za pomocą prostych logicznych warunków – np. kamera wykrywająca ruch, która włącza żarówkę w pokoju lub ogrzewanie włączające się jeżeli temperatura spadnie poniżej 20°C.
